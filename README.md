# Portfolio Profesional - Edinson Medina Chinga

Este proyecto es un CV digital premium construido con **Astro 5** y **Tailwind CSS 4**. Está diseñado para ser rápido, accesible y altamente profesional.

## 🚀 Tecnologías

- **Core:** Astro 5 (Última versión estable)
- **Estilos:** Tailwind CSS 4 (Vite-based)
- **Iconos:** @lucide/astro
- **Tipografía:** Inter (Google Fonts)
- **Lenguaje:** TypeScript

## 📂 Estructura

- `src/components/`: Componentes modulares de la interfaz.
- `src/data/`: Archivos TypeScript con el contenido del CV para fácil mantenimiento.
- `src/layouts/`: Estructura base HTML y configuración SEO.
- `src/styles/`: Estilos globales y configuración de temas.

## 🛠️ Instalación y Desarrollo

1. **Instalar dependencias:**

   ```bash
   npm install
   ```

2. **Iniciar servidor de desarrollo:**

   ```bash
   npm run dev
   ```

3. **Construir para producción:**

   ```bash
   npm run build
   ```

## ✨ Características

- **Single Page:** Navegación suave entre secciones.
- **Scroll Spy:** Resaltado automático de la sección activa en el Navbar.
- **Modo Oscuro:** Persistente mediante `localStorage`.
- **SEO Ready:** Meta tags optimizados y estructura semántica.
- **Responsive:** Diseño adaptado a móviles, tablets y escritorio.
